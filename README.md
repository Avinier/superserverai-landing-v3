# SuperServer AI

AI sandbox infrastructure platform - deploy, manage, and scale your AI workloads with enterprise-grade infrastructure.

## Tech Stack

- **Framework**: React 19
- **Build Tool**: Vite 6
- **Styling**: Tailwind CSS 4
- **Runtime**: Bun
- **Language**: TypeScript

## Getting Started

### Prerequisites

- [Bun](https://bun.sh/) (v1.0 or higher)

### Installation

```bash
# Install dependencies
bun install

# Start development server
bun dev

# Build for production
bun run build

# Preview production build
bun preview
```

## Design System

### Colors

| Name | Value | Usage |
|------|-------|-------|
| Primary | `#254bf1` | Buttons, links, accents |
| Secondary | `#FF3C5B` | Highlights, alerts |
| Background | `#0a0a0a` | Page background |
| Surface | `#141414` | Cards, sections |
| Surface Elevated | `#1a1a1a` | Hover states, modals |
| Text | `#fefdfa` | Primary text |
| Text Muted | `#a0a0a0` | Secondary text |
| Grey | `#34322d` | Subtle elements |
| Border | `#2a2a2a` | Dividers, outlines |

### Typography

- **Tiempos**: Headlines and titles
- **Geist**: Body text and UI elements
- **Montserrat**: Backup/alternative
